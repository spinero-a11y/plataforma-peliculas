# 🎬 CineFlex - Plataforma de Catálogo de Películas

¡Bienvenido a **CineFlex**! Una aplicación web interactiva inspirada en interfaces modernas de streaming como Netflix o HBO, pero diseñada con una paleta de colores cálida, oscura y acogedora. 

Este proyecto fue desarrollado utilizando **Visual Studio** y está conectado directamente con este repositorio de **GitHub**.

---

## 🚀 Características Principales

*   **Buscador Inteligente:** Filtra películas en tiempo real conforme vas escribiendo, sin necesidad de recargar la página.
*   **Diseño Premium Oscuro:** Interfaz moderna que utiliza tonos oscuros combinados con acentos cálidos (dorado/ámbar) para una experiencia visual cómoda y cinematográfica.
*   **Navegación Intuitiva:** Menú superior con secciones para *Inicio*, *Recomendados*, *Mi Lista* y un menú desplegable dinámico para explorar por categorías.
*   **Sliders Deslizables:** Las películas se organizan en filas horizontales scrolleables, imitando la experiencia de las televisiones inteligentes y apps móviles de streaming.
*   **Conexión con API Real:** Consumo de datos en tiempo real (pósters, títulos, puntuaciones y sinopsis) a través de la API de **The Movie Database (TMDb)**.

---

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica de la plataforma.
*   **CSS3:** Estilos personalizados, transiciones suaves, diseño adaptativo y variables de entorno para la paleta de colores.
*   **JavaScript (ES6+):** Lógica del buscador, peticiones asíncronas (`fetch`, `async/await`) y manipulación dinámica del DOM.
*   **TMDb API:** Fuente de datos externa para el catálogo cinematográfico.
*   **Git & GitHub:** Control de versiones y alojamiento del código.
